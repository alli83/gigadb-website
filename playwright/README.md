# Playwright

## Run tests locally

- First, make sure project is running locally on [http://gigadb.gigasciencejournal.com:9170](http://gigadb.gigasciencejournal.com:9170)
- Have `pnpm` installed: [https://pnpm.io/installation](https://pnpm.io/installation)

```sh
# cd to this folder
cd playwright
# install deps
pnpm i
# Run all tests
pnpm test
```
